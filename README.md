# Superstore Sales Analysis (Advanced Excel)

An advanced Excel project analyzing superstore sales data to identify trends, top-performing products, regional performance, and actionable business insights. 

## Project Overview
This project focuses on analyzing a superstore sales dataset using **advanced Excel techniques** including PivotTables, charts, dashboards, and formulas. The goal is to provide data-driven insights for better business decisions.

## Features
- Comprehensive **sales trend analysis** over time
- **Top-selling products** and categories identification
- **Regional performance analysis** to spot high and low performing areas
- **Interactive Excel dashboard** for quick insights
- Use of **PivotTables, VLOOKUP, conditional formatting, and charts**

## Technology Used
- Microsoft Excel (Advanced)
- PivotTables, Charts, Dashboards, Formulas
